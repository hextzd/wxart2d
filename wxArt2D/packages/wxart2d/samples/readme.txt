Here is where all modules or stored.

canvas: 

the core canvas library
including tools for editing.

A range of samples and tutorial samples are available.

editor:

editor using the tools from the core, an editor frame and some dialogs.
The samples demonstrate a single frame editor and a multi frame editor.

gdsio:

Input and output handler for GDSII or calma stream format, used in IC/chips
development.
( in the making )

svgio:

Input and output handler for SVG format
